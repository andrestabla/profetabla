Coloca aquí los archivos CSV para la carga del módulo "Ocupaciones y habilidades del Siglo XXI".

Nombres esperados:
- `Employment_occupation*.csv` (fuentes Europa, separadas por `;`)
- `National Employment Matrix_IND_*.csv` (matrices EE.UU. con código de industria en el nombre)

Ejemplos:
- `Employment_occupation_es.csv`
- `National Employment Matrix_IND_541500.csv`

Nota:
- Las cifras de Europa se normalizan a miles de empleos (`/1000`).
- Las cifras de EE.UU. ya se interpretan en miles de empleos.
